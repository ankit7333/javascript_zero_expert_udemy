JavaScript Fundamentals – Part 1

let

  • The let declaration declares a block-scoped local variable
  • The value of a let can be changed through reassignment

const

  • The const declaration creates block-scoped constants.
  • The value of a constant can't be changed through reassignment
  • and it can't be redeclared
  • while using const we need initializing value

var

The var statement declares a function-scoped or globally-scoped variable, optionally initializing it to a value.
