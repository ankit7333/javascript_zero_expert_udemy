JavaScript Fundamentals – Part 1

Data Types

  • Primitive (7 types of Data Types)

    JavaScript primitive data types are data types that refer to a single value.


    • 1. Number (FLoating point number) :- used for decimals and integers i.e let age = 18;

    • 2. String (Sequence of characters):- used for text i.e let firstName = 'Rabindra';

    • 3. Boolean (Logical type that can be true or false):- used for taking decisions i.e let fullAge = true;

    • 4. Undefined :- value taken by a variable that is not yet define ('empty value') i.e let children;

    • 5. Null :- also means empty value i.e let children;

    • 6. Symbol (ES2015) :- Value that is unique and cannot be changed Not useful for now

    • 7. BigInt (ES2020) :- Larger interger than the Number type can hold

  • Non - Primitive

    JavaScript non-primitive types are objects. An object holds a reference/address of a single key-value pair or many key-value pairs.

    Object e.g :-
    let me = {
      firstName:'Rabindra',
      lastName:'Negi'
    }
