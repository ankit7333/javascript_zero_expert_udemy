Constructor Functions and other Operator

Prototypes

Every object in JavaScript has a built-in property, which is called its prototype.

Prototypal Inheritance

- is a feature in javascript used to add methods and properties in objects.

- a method by which an object can inherit the properties and methods of another object.

The Prototype Chain

- All objects in JavaScript travels back to Object's prototype

- Which means every object in JavaScript is created from Object constructor.

ES6 Classes

Classes are a template for creating objects.

2 types of classes

- class expression

- class declaration

Setters

The set syntax binds an object property to a function to be called when there is an attempt to set that property.

Getters

The get syntax binds an object property to a function to be called that property is looked up.

Object.create

The Object.create() method creates a new object, using an existing object as the prototype of the newly created object.

super()

The super keyword is used to call the constructor of its parent class to access the parent's properties and methods.

Encapsulation: Protected Properties

Protected properties are usually prefixed with an underscore _.

Encapsulation: Private Class Fields

Private class members can be created by using a hash # prefix.
